# Отчет по заданию: Инвертирование формальных степенных рядов

## 1. Цель работы

Реализация алгоритма нахождения первых $n$ коэффициентов обратного элемента в кольце формальных степенных рядов $R[[x]]$ над полем вещественных чисел $\mathbb{R}$. Проверка корректности результата через вычисление произведения Коши.

## 2. Теоретическое обоснование

### 2.1. Критерий обратимости

Ряд $f = \sum_{i=0}^{\infty} a_i x^i$ обратим в кольце $R[[x]]$ тогда и только тогда, когда его свободный член $a_0$ обратим в кольце $R$. В случае $R = \mathbb{R}$ это означает условие $a_0 \neq 0$.

### 2.2. Рекуррентная формула

Пусть $f^{-1} = g = \sum_{k=0}^{\infty} b_k x^k$. Из условия $f \cdot g = 1$ следует система уравнений:

1. Для $k=0$: $a_0 b_0 = 1 \implies b_0 = a_0^{-1}$.
2. Для $k > 0$: $\sum_{i=0}^k a_i b_{k-i} = 0$.

Отсюда выводится формула для последовательного нахождения коэффициентов:
$$b_k = -\frac{1}{a_0} \sum_{j=1}^k a_j b_{k-j}$$

### 2.3. Произведение Коши

Для проверки результата используется произведение двух рядов (многочленов), ограниченное точностью $n$ (работа в кольце $R[x]/(x^n)$). Коэффициент $c_k$ произведения вычисляется как:
$$c_k = \sum_{j=0}^k a_j b_{k-j}$$

## 3. Описание реализации

Программа включает следующие ключевые компоненты:

* **`get_inverse`**: Функция, реализующая рекуррентное вычисление коэффициентов $b_k$. Включает проверку обратимости ($a_0 \neq 0$) и автоматическое дополнение входного многочлена нулями до нужной степени $n$.
* **`get_coef_multiple`**: Оптимизированная функция вычисления произведения Коши с ограничением по степени (`limit`). Сложность алгоритма составляет $O(n \cdot \text{limit})$.
* **`format_series`**: Вспомогательная функция для представления рядов в удобочитаемом математическом виде ($a_0 + a_1x + a_2x^2 \dots$).
* **`is_unit`**: Функция верификации, проверяющая, совпадает ли полученное произведение с единичным рядом $1 + 0x + 0x^2 + \dots$ с учетом погрешности вычислений с плавающей точкой.

## 4. Тестирование

### 4.1. Пример 1: Частичная сумма экспоненты

Рассматривался ряд $f = \sum_{k=0}^{10} \frac{x^k}{k!}$, представляющий собой первые 11 членов разложения $e^x$.

* **Ожидаемый результат**: Поскольку $(e^x)^{-1} = e^{-x}$, коэффициенты обратного ряда должны соответствовать разложению $e^{-x}$ в ряд Тейлора (знакочередующийся ряд $\frac{(-1)^k}{k!}$).
* **Результат программы**: Получен ряд $1.0 - 1.0x + 0.5x^2 - 0.1667x^3 \dots$, что полностью соответствует теории. Проверка произведения по модулю $x^{10}$ подтвердила корректность.

### 4.2. Пример 2: Многочлен $f = x^2 - x - 1$

Коэффициенты: $a_0 = -1, a_1 = -1, a_2 = 1$.

* **Результат программы**: Найдены первые 10 коэффициентов обратного ряда. Произведение $f \cdot f^{-1} \pmod{x^{10}}$ дало единичный ряд.

## 5. Выводы

1. Алгоритм рекуррентного вычисления коэффициентов эффективно находит обратный элемент для любого ряда с ненулевым свободным членом.
2. Использование типа `float` требует введения функции `is_close` для корректной проверки равенства коэффициентов нулю из-за накопления ошибок округления при делении.
3. Ограничение вычислений произведения пределом `limit` позволяет эффективно проверять результаты в рамках заданного порядка точности $n$, имитируя работу в кольце вычетов по модулю идеала $(x^n)$.
