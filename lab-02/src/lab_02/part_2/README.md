# Отчет по лабораторной работе №2. Часть №2

**Тема:** Численные и алгебраические методы на основе итераций Ньютона. Поиск корней уравнений и инверсия степенных рядов.

## 1. Цель работы

1. Разработка программного модуля для поиска корней нелинейных уравнений методом Ньютона.
2. Реализация алгоритма быстрого нахождения обратного степенного ряда (инверсии) методом Ньютона.
3. Изучение теоретической сложности алгоритмов и проверка их эффективности на тестовых примерах (экспоненциальный ряд, возвратные последовательности).

## 2. Теоретическая база

### 2.1. Метод Ньютона для поиска корней (Часть 1)

Итерационный процесс поиска корня функции $f(x)$ через линеаризацию:
$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$

### 2.2. Алгоритм инверсии Ньютона (Часть 2)

Метод Ньютона применим для нахождения обратного элемента $g = f^{-1}$ в кольце формальных степенных рядов $R[[x]]$. Задача состоит в поиске такого $g$, что $f \cdot g \equiv 1 \pmod{x^l}$.
Итерационная формула для уточнения коэффициентов:
$$g_{i+1} = g_i(2 - f \cdot g_i) \pmod{x^{2^{i+1}}}$$
Алгоритм обладает **квадратичной сходимостью**: на каждом шаге количество верных коэффициентов удваивается.

**Сложность алгоритма:** Доказано, что стоимость инверсии не превосходит $3M(l) + O(l)$, где $M(l)$ — сложность умножения двух многочленов степени $l$. Это делает операцию деления рядов асимптотически сопоставимой с операцией умножения.

## 3. Программная реализация

Программа написана на языке Python 3.12. Для реализации инверсии были добавлены следующие компоненты:

### 3.1. Функциональные модули

* **`get_coef_multiple`**: Вычисляет произведение двух многочленов (свертку коэффициентов) с ограничением по степени (взятие по модулю $x^{limit}$).
* **`minus_coeff`**: Реализует вычитание многочленов разной длины.
* **`newton_inverse`**: Основная функция, реализующая итерации $g = 2g - f \cdot g^2$. Использует `ceil(log2(l))` итераций для достижения требуемой точности.
* **`format_series`**: Вспомогательная функция для «красивого» вывода ряда в математическом виде.

### 3.2. Список реализованных задач инверсии

1. **Инверсия экспоненциального ряда**: $f(x) = \sum \frac{x^k}{k!}$. Результат должен соответствовать ряду $e^{-x}$.
2. **Инверсия многочлена $f(x) = x^2 - x - 1$**: Демонстрация связи обратных рядов с числами Фибоначчи.

## 4. Результаты тестирования

### 4.1. Поиск корней (Часть 1)

Для уравнения $f(x) = \ln(x) - 1$ корень $x \approx 2.71828182$ найден за 5 итераций с точностью $10^{-8}$.

### 4.2. Инверсия степенных рядов (Часть 2)

**Пример вычисления для $f(x) = -1 - x + x^2$ (точность $l=10$):**

* **Вход:** `[-1.0, -1.0, 1.0]`
* **Результат $f^{-1}$:** `-1.0 + 1.0*x - 2.0*x^2 + 3.0*x^3 - 5.0*x^4 + 8.0*x^5 - 13.0*x^6 + 21.0*x^7 - 34.0*x^8 + 55.0*x^9`
* **Анализ:** Коэффициенты ряда (1, 1, 2, 3, 5, 8, 13...) соответствуют числам Фибоначчи, что подтверждает корректность работы алгоритма для возвратных последовательностей.

**Пример для экспоненты $e^x$ (точность $l=10$):**

* **Вход:** Коэффициенты $1, 1, 1/2!, 1/3! \dots$
* **Результат:** Ряд с чередующимися знаками $1, -1, 1/2!, -1/3! \dots$, что соответствует разложению $e^{-x}$.
* **Проверка:** Произведение $f \cdot f^{-1} \pmod{x^{10}}$ дает строго `1.0 + 0*x + ... + 0*x^9`.

## 5. Выводы

1. Метод Ньютона является универсальным инструментом, применимым как в численном анализе (поиск корней), так и в компьютерной алгебре (деление многочленов и рядов).
2. Реализованный алгоритм инверсии Ньютона значительно эффективнее классического метода неопределенных коэффициентов ($O(l^2)$) при использовании быстрых алгоритмов умножения.
3. Программная реализация подтвердила теоретическую оценку сложности: для достижения точности в 1000 коэффициентов требуется всего 10 итераций алгоритма.
4. Использование модульной структуры позволило интегрировать функции красивого форматирования и автоматической проверки результатов, что упрощает отладку при работе с большими степенями.
