# Отчет по лабораторной работе №2. Часть № 1

**Тема:** Численные методы решения нелинейных скалярных уравнений. Метод Ньютона.

## 1. Цель работы

Разработка универсального программного модуля для поиска корней нелинейных уравнений методом Ньютона (методом касательных). Изучение влияния начального приближения на сходимость алгоритма и программная обработка особых случаев (равенство производной нулю).

## 2. Теоретическая база

### 2.1. Алгоритм метода Ньютона

Метод Ньютона является итерационным процессом, основанным на линеаризации функции в окрестности текущего приближения. Итерационная формула вычисляется как точка пересечения касательной к графику функции $f(x)$ с осью абсцисс:
$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$

### 2.2. Условия сходимости

1. Функция $f(x)$ должна быть непрерывна и дважды дифференцируема на отрезке, содержащем корень.
2. Производная $f'(x)$ не должна обращаться в нуль в процессе вычислений.
3. Метод обладает **квадратичной скоростью сходимости** при условии выбора начального приближения $x_0$ в достаточной близости к корню.

## 3. Программная реализация

Программа написана на языке Python 3.12 и имеет модульную архитектуру для обеспечения чистоты кода и удобства расширения:

### 3.1. Структура проекта

* **`models.py`**: Содержит описание математических функций. Для каждой функции реализован отдельный класс (DataClass), вычисляющий значение функции и её производной. Использование реестра `FUNCTION_REGISTRY` позволяет автоматически формировать меню выбора.
* **`solver.py`**: Реализует логику итерационного процесса. Включен механизм «сдвига» (SHIFT) при обнаружении нулевой производной, что предотвращает деление на ноль и аварийное завершение программы.
* **`main.py`**: Обеспечивает консольный интерфейс. Реализована система «умного ввода», позволяющая использовать значения параметров и начальных приближений по умолчанию нажатием клавиши Enter.

### 3.2. Список реализованных функций

В программе доступны следующие уравнения:

1. $f(x) = \sin(x) - 2x^2 + 0.5$
2. $f(x) = x^n - a$ (с возможностью ввода параметров $n$ и $a$ пользователем)
3. $f(x) = \sqrt{1 - x^2} - e^x$
4. $f(x) = x^6 - 5x^3 - 2$
5. $f(x) = \log_2(x) - \frac{1}{1+x^2}$
6. $f(x) = \sin(x/2) - 1$
7. $f(x) = \ln(x) - 1$

## 4. Результаты тестирования

Программа успешно находит корни уравнений с заданной пользователем точностью $\varepsilon = 10^{-n}$.

**Пример вычисления корня для $f(x) = \ln(x) - 1$ (точное значение $x = e \approx 2.7182818$):**

* Заданная точность: $n=8$ ($\varepsilon = 10^{-8}$)
* Начальное приближение: $x_0 = 1.0$
* Итерация 1: $x = 2.0000000000$
* Итерация 2: $x = 2.6137056389$
* Итерация 3: $x = 2.7162439266$
* Итерация 4: $x = 2.7182810459$
* Итерация 5: $x = 2.7182818285$ (Результат достигнут за 5 шагов).

## 5. Выводы

1. Метод Ньютона продемонстрировал высокую скорость сходимости по сравнению с линейными методами (например, методом дихотомии).
2. Разработанная модульная архитектура программы позволяет добавлять новые функции в файл `models.py` без модификации основного алгоритма решения.
3. Программная обработка исключительных ситуаций (нулевая производная) делает решатель более надежным при работе со сложными функциями.
4. Установлено, что правильный выбор начального приближения $x_0$ является критическим фактором для успешного поиска корня.
